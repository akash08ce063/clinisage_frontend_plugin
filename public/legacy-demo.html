How to Use

To embed this widget on your website, add the following code to your HTML:

<!-- Widget Configuration -->
<script>
    window.clinisageConfig = {
        authToken: 'YOUR_AUTH_TOKEN_HERE',
        agentName: 'Medical Assistant',
        themeColor: '#0ea5e9',
        position: 'bottom-right'
    };
</script>

<!-- Widget Script -->
<script src="https://your-site.netlify.app/widget.js" async></script>

Configuration Options

  • authToken - Your authentication token (required)
  • agentName - Display name for the assistant
  • themeColor - Primary color (hex format)
  • backgroundColor - Widget background color
  • textColor - Text color
  • position - Widget position: 'bottom-right', 'bottom-left', 'top-right', 'top-left'
